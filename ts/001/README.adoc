= TS-1: Guiding Principles for Software Development
:toc: macro
:toc-title: Contents

This technical standard offers a philosophical framework for approaching software development projects. It outlines key principles for software design and construction that, in combination, are proven to lead to successful outcomes.

Related technical standards include:

* *link:../010/README.adoc[TS-10: System Design]*, which establishes some key principles that underpin good software architecture.
* *link:../012/README.adoc[TS-12: Application Architecture]*, which covers design patterns for standalone applications and services.
* *link:../013/README.adoc[TS-13: Code Design]*, which covers more low-level concerns such as code formatting and local design patterns.
